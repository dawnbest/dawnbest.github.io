This website was created to showcase the webcomic, Sylvanna. It's a basic CMS site running Wordpress with a few HTML add-ons.

Objectives

Overview: A CMS website for the Sylvanna webcomic, featuring art, stories, and the comic itself. This site was created in Wordpress and uses various Wordpress plugins for its multiple galleries. Some custom HTML was also added to sidebars and other locations, for easier navigation.
